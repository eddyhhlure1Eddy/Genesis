## For New Users

Start here:
1. Read [QUICKSTART_WEBUI.md](QUICKSTART_WEBUI.md) or [QUICK_START_CN.md](QUICK_START_CN.md)
2. Follow the installation steps
3. Run `start_webui_integrated.bat`
4. Refer to [WEBUI_GUIDE.md](WEBUI_GUIDE.md) for feature details

## Getting Help

If you encounter issues:
1. Check [TROUBLESHOOTING.md](TROUBLESHOOTING.md) first
2. Review relevant configuration guides
3. Ensure nested Python environment is correctly configured (see [PYTHON_ENV.md](PYTHON_ENV.md))
